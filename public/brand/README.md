# AIGround vClaude 브랜드 자산

이 폴더는 기존 `brand-assets` 구조를 그대로 유지하면서,
색상만 Claude/Anthropic 계열의 분위기로 재해석한 버전입니다.

주의:

- 공개된 공식 브랜드 가이드를 찾지 못해, 공식 웹페이지의 인상에서 추론한 팔레트입니다.
- 적용 기준일: 2026-04-13
- 구조, 비율, 배치는 기존 자산과 동일하고 색상만 변경했습니다.

적용한 색상 방향:

- 메인 다크: `#241B16`
- 다크 배경: `#1B1512`
- 라이트 배경: `#F7F3EE`
- 보조 라이트 배경: `#E9DED2`
- 포인트 컬러: `#C96E4B`
- 본문/보조 텍스트: `#3E2F27`, `#8B7768`, `#A08C7C`

현재 파일:

- `app-icon-light.svg`
- `app-icon-dark.svg`
- `favicon-light.svg`
- `favicon-dark.svg`
- `logo-primary-light.svg`
- `logo-primary-dark.svg`
- `logo-stacked-light.svg`
- `logo-stacked-dark.svg`
- `logo-mono-black.svg`
- `logo-mono-white.svg`
- `logotype-wordmark-light.svg`
- `logotype-wordmark-dark.svg`
- `logotype-watermark-black.svg`
- `logotype-watermark-white.svg`

권장 사용:

- 라이트 모드 UI: `logo-primary-light.svg`, `app-icon-light.svg`, `favicon-dark.svg`
- 다크 모드 UI: `logo-primary-dark.svg`, `app-icon-dark.svg`, `favicon-light.svg`
- 세로형 배치: `logo-stacked-light.svg`, `logo-stacked-dark.svg`
- 단색 인쇄물 또는 오버레이: `logo-mono-black.svg`, `logo-mono-white.svg`
- 타이포 중심 브랜딩: `logotype-wordmark-light.svg`, `logotype-wordmark-dark.svg`
- 워터마크 또는 서명형 사용: `logotype-watermark-black.svg`, `logotype-watermark-white.svg`
